## SEO Tips for Content Creation

### When Writing Blog Posts:
1. **Title** - Include target keyword (first 60 characters)
   - ✅ Good: "Best SCADA Systems for Manufacturing in Kenya | AutoNex"
   - ❌ Bad: "Our Latest Blog Post"

2. **Meta Description** - 150-160 characters, include keyword
   - Should match what you want to say in search results
   - Include call-to-action (Learn more, Get quote, etc.)

3. **Headings** - Use H1 once, natural keyword placement
   - H1: Main topic title
   - H2: Section headers (include related keywords)
   - H3: Subsections

4. **Content** - 2000+ words for competitive keywords
   - Include LSI keywords (related terms)
   - Add images with descriptive ALT text
   - Use bullet points and tables
   - Include internal links

5. **Front Matter** - Add slug and metadata:
   ```
   ---
   title: "SCADA Systems for Manufacturing Kenya"
   slug: "scada-manufacturing-kenya"
   description: "Professional SCADA solutions..."
   ---
   ```
